📝 Resumen de Endpoints

  • 📌 GET /superheroes → Devuelve todos los superhéroes de la base de datos
  • 📌 GET /superheroes/:nombre → Busca un superhéroe por su nombre exacto
  • ✅ 200 → Respuesta exitosa con datos en JSON
  • ⚠️ 404 → Superhéroe no encontrado
  • ❌ 500 → Error interno del servidor
